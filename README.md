# Tichi Login Automation

## Project Description
This project automates the login functionality of the Tichi web application using Python and Selenium WebDriver.

## Technologies Used
- Python
- Selenium
- WebDriver Manager
- PyTest

## Project Structure

```
Tichi_Project/
│
├── login.py
├── test.py
├── requirements.txt
└── README.md
```

## Website
https://tichi-app-webapp-stage.web.app/login

## Run

```bash
pip install -r requirements.txt
python login.py
```
